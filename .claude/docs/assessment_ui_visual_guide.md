## Color Scheme

### Primary Colors
- **Active/Selected**: `slate-900` (black)
- **Text**: `slate-900` for primary, `slate-600` for secondary
- **Borders**: `slate-200` for default, `slate-900` for active

### Status Colors
- **Success/Correct**:
  - Background: `green-50`, `green-100`
  - Border: `green-200`
  - Text: `green-600`, `green-900`
  - Icon: `green-600`

- **Error/Incorrect**:
  - Background: `red-50`, `red-100`
  - Border: `red-200`
  - Text: `red-600`, `red-900`
  - Icon: `red-600`

- **Warning/Review**:
  - Background: `amber-50`
  - Border: `amber-200`
  - Text: `amber-600`, `amber-900`
  - Icon: `amber-600`

### Interactive States
- **Default**: `border-slate-200 hover:border-slate-300`
- **Selected**: `border-slate-900 bg-slate-50 ring-2 ring-slate-900`
- **Disabled**: `opacity-50 pointer-events-none`

## Typography Scale

- **Page Title**: `text-3xl font-bold` (Manual Selection)
- **Section Title**: `text-2xl font-bold` (Test header)
- **Card Title**: `text-xl font-semibold` (Manual cards)
- **Question Text**: `text-lg font-medium` (Questions)
- **Body Text**: `text-base` (Answers, descriptions)
- **Small Text**: `text-sm` (Labels, metadata)
- **Extra Small**: `text-xs` (Badges in results)

## Spacing Guidelines

- **Container Padding**: `px-4 py-8`
- **Card Padding**: `p-6` (CardHeader), `p-4` (CardContent)
- **Element Gaps**: `gap-2`, `gap-3`, `gap-4`, `gap-6`
- **Section Margins**: `mb-6`, `mb-8`
- **Card Grid Gaps**: `gap-6`

## Responsive Breakpoints

### Grid Layouts
- **Mobile** (`default`): 1 column
- **Tablet** (`md:`): 2 columns
- **Desktop** (`lg:`): 3 columns

### Question Grid
- **Mobile**: 8 columns (8x5 grid)
- **Tablet+** (`sm:`): 10 columns (10x4 grid)

### Container Widths
- Manual Selection: Full width with `container mx-auto`
- Test Taking: `max-w-4xl mx-auto`
- Results: `max-w-5xl mx-auto`

## Animation & Transitions

- **Hover Effects**: `transition-all` on buttons and cards
- **Shadow**: `hover:shadow-lg` on manual cards
- **Loading States**:
  - Spinner: `animate-spin` (Loader2 icon)
  - Progress bar: `animate-pulse`
- **Dialog**: Built-in animations from Radix UI
  - Fade in/out
  - Zoom in/out
  - Slide animations

## Icon Usage

### Lucide React Icons
- **BookOpen**: Manual cards, study recommendations
- **Award**: Passed status
- **TrendingUp**: Failed status (improvement needed)
- **CheckCircle2**: Correct answers
- **XCircle**: Incorrect answers
- **ChevronLeft/Right**: Navigation
- **ChevronDown/Up**: Expand/collapse
- **AlertCircle**: Warnings and errors
- **RotateCcw**: Retry/new assessment
- **X**: Close/exit
- **Loader2**: Loading states

## Accessibility Features

- **Focus Rings**: `focus-visible:ring-2 focus-visible:ring-slate-950`
- **Screen Reader Text**: `sr-only` class for context
- **ARIA Labels**: Built into shadcn/ui components
- **Keyboard Navigation**: Tab through all interactive elements
- **Color Contrast**: WCAG AA compliant
- **Semantic HTML**: Proper heading hierarchy

## Mobile Optimizations

- Touch-friendly buttons: Minimum 44px height
- Larger text on questions: `text-lg`
- Generous padding on answer options: `p-4`
- Full-width buttons on mobile
- Responsive grid that collapses to single column
- Stack action buttons vertically on small screens
- Adequate spacing between interactive elements
